# 欧盟 CE 认证流程与标志规范

## 概述

CE 标志是产品进入欧盟市场的强制性合规标志，表示产品符合欧盟相关指令的基本要求。涵盖电子电器、玩具、机械、医疗器械、个人防护设备等多类产品。

## 认证流程

1. **确认适用指令**：根据产品类型确定适用的欧盟指令（如玩具 EN 71、低压设备 LVD、电磁兼容 EMC）
2. **进行风险评估**：按照指令要求进行产品风险评估
3. **测试与符合性评估**：
   - 低风险产品：自我声明（Module A）
   - 中高风险产品：须由公告机构（Notified Body）进行评估
4. **编制技术文件**：包含产品设计图、测试报告、风险评估、使用说明
5. **签署符合性声明**（Declaration of Conformity）
6. **加贴 CE 标志**：标志须清晰、可见、不可擦除，最小高度 5mm

## CE 标志规范

- 标志由"CE"两个字母组成，E 的内切线为 C 下半部分的延伸
- 须贴在产品本体或铭牌上，也可贴在包装或随附文件上
- 如有公告机构参与，须在 CE 标志后标注机构编号
- 不得与其他标志混淆或误导

## 常见指令

- 玩具安全指令 2009/48/EC（对应 EN 71 标准）
- 低压设备指令 2014/35/EU（LVD）
- 电磁兼容指令 2014/30/EU（EMC）
- 无线电设备指令 2014/53/EU（RED）
- 机械指令 2006/42/EC
