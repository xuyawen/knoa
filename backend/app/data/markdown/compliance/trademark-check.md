# 商标侵权排查清单

## 商标侵权风险

亚马逊对商标侵权执行零容忍政策。使用他人注册商标作为产品名称、描述、关键词，或销售带有他人商标的产品，均可能触发侵权投诉。后果包括 listing 下架、账户冻结、资金冻结 90 天。

## 排查流程

1. **品牌备案查询**：在亚马逊 Brand Registry 确认目标品牌是否已备案
2. **USPTO 查询**：在美国专利商标局（USPTO）TESS 系统搜索商标注册状态
3. **EUIPO 查询**：在欧盟知识产权局查询欧洲商标
4. **类目匹配**：确认商标注册类别是否覆盖你的产品类目（尼斯分类）
5. **关键词审查**：listing 标题、五点描述、Search Terms、A+ 页面全部审查

## 高风险场景

- 在标题或描述中使用竞品品牌名作为关键词
- 销售兼容配件时使用"for iPhone""for Samsung"等表述（应使用"compatible with"）
- 产品上印有未授权的品牌 Logo
- 域名或店铺名包含他人商标

## 合规做法

- 兼容产品使用"compatible with [brand] [model]"而非"for [brand]"
- 自己注册商标并完成亚马逊品牌备案，保护自有品牌
- 定期用 Brand Registry 的 Search Counterfeit 工具监控侵权
- 收到投诉后及时申诉，提供授权书或不侵权证明
