# 外观专利侵权排查清单

## 为什么要排查

外观专利侵权是亚马逊下架 listing 的常见原因之一。美国 USPTO 外观专利保护期 15 年，欧盟 EUIPO 保护期最长 25 年。侵权后果包括 listing 下架、账户冻结、库存没收、甚至法律诉讼。

## 排查流程

1. **产品拍照**：拍摄产品六面图（正、背、左、右、顶、底）及使用状态图
2. **反向图片检索**：使用 Google Patents、USPTO Patent Public Search 进行图片反向检索
3. **关键词检索**：用产品功能描述词在 USPTO 数据库搜索外观专利（Design Patent）
4. **对比分析**：重点对比整体轮廓、比例、表面图案、装饰元素，判断是否"实质相似"
5. **记录留档**：排查过程和结论须记录留档，作为尽调证据

## 重点排查对象

- 3C 配件（手机壳、支架、充电器外观极易撞专利）
- 家居装饰品（灯具、花瓶、摆件）
- 厨具餐具（刀型、杯型、盘型）
- 包装设计（盒型、图案）

## 应对策略

- 选品阶段优先做专利排查，而非上架后补救
- 发现相似专利时，做差异化改良（改变轮廓比例、表面纹理）
- 收到侵权投诉时，可申请专利无效或提供不侵权意见书
- 建议保留供应商的设计原图或委托设计合同，作为抗辩证据
