# 海外仓与 FBA 对比选择

## 两种仓储模式

跨境电商卖家有两种主流仓储选择:亚马逊 FBA(Fulfillment by Amazon)和第三方海外仓(Third-party Warehouse)。两者各有优劣,选择哪种取决于产品特性、销售策略、成本结构。

**FBA**:商品存放在亚马逊仓库,由亚马逊负责拣货、包装、配送、客服、退货。买家下单后亚马逊 Prime 会员可享 1-2 天送达。

**海外仓**:商品存放在第三方海外仓,由海外仓负责拣货、包装、配送,卖家自行处理客服和退货。配送时效 2-5 天,略慢于 FBA。

## FBA 优势与劣势

**优势**:

- **Prime 标志**:FBA 商品带 Prime 标志,吸引 Prime 会员(美国有 2 亿+ Prime 会员),转化率提升 20%-40%。
- **Buy Box 优先**:同等条件下,FBA 卖家更容易赢得 Buy Box。
- **配送时效**:1-2 天送达,客户体验最佳。
- **客服外包**:亚马逊负责售后客服和退货处理,卖家省心。
- **流量倾斜**:亚马逊算法对 FBA 商品有流量倾斜,自然排名更靠前。

**劣势**:

- **费用高**:仓储费 + 配送费 + 佣金,总费用占售价 30%-50%。
- **规则严格**:标签错误、包装不合规会被拒收入库或收费重新包装。
- **库存风险**:长期仓储费、滞销品处理成本高。
- **退货不可控**:亚马逊退货政策宽松,退货率高,且退货商品可能无法二次销售。
- **资金占用**:库存压在亚马逊仓库,资金周转慢。

## 海外仓优势与劣势

**优势**:

- **费用灵活**:仓储费通常比 FBA 低 20%-40%,无长期仓储费。
- **规则灵活**:可以自定义包装、放营销卡片、组合捆绑,FBA 不允许放非亚马逊物料。
- **多平台共用**:一个海外仓可以服务亚马逊、eBay、Walmart、Shopify 等多平台,FBA 只能服务亚马逊。
- **退货可控**:退货回到海外仓,可以检查、翻新、二次销售,减少损失。
- **库存灵活**:可以随时调货到 FBA 补充库存,或调回国内处理。

**劣势**:

- **无 Prime 标志**:配送时效 2-5 天,转化率低于 FBA。
- **Buy Box 劣势**:同等条件下,海外仓卖家 Buy Box 胜率低于 FBA 卖家。
- **需自行处理客服**:客服和退货处理需要卖家自行或委托第三方,人力成本高。
- **配送成本**:单件配送费可能高于 FBA(因为 FBA 有规模效应)。
- **管理复杂**:需要对接海外仓系统,管理库存调拨、订单同步、退货运回。

## 选择决策矩阵

根据以下场景选择:

**选 FBA 的场景**:

- 亚马逊单平台销售,且客单价 > 15 美元(FBA 费用占比可控)。
- 标准件商品(非大件、非危险品)。
- 追求 Prime 标志和 Buy Box 优势。
- 新品期需要快速积累 Review 和权重。
- 卖家没有海外团队处理客服和退货。

**选海外仓的场景**:

- 多平台销售(亚马逊 + eBay + Walmart + Shopify)。
- 大件商品(FBA 大件配送费极高,海外仓更划算)。
- 客单价 < 15 美元(FBA 费用占比过高)。
- 退货率高(如服装、鞋类),需要翻新二次销售。
- 需要自定义包装或放营销物料。
- 滞销品需要暂存,等待清仓时机。

**混合模式(推荐)**:

成熟卖家通常用 FBA + 海外仓混合模式:

- **主力 SKU 走 FBA**:80% 的库存发到 FBA,享受 Prime 标志和 Buy Box 优势。
- **滞销品移到海外仓**:库龄超过 90 天的 FBA 库存移到海外仓暂存,避免长期仓储费。
- **多平台订单走海外仓**:eBay、Walmart 等非亚马逊订单从海外仓发货。
- **退货处理走海外仓**:FBA 退货移到海外仓,检查翻新后重新发回 FBA 或在二手平台销售。
- **紧急补货从海外仓调货**:FBA 断货时,从海外仓调货到 FBA,比从国内发快 2-3 周。

## 成本对比示例

以美国站标准件商品(客单价 25 美元,重量 0.5kg)为例:

**FBA 成本**:

- 配送费:3.06 美元
- 仓储费(月):0.5 美元
- 佣金(15%):3.75 美元
- 月度总成本:7.31 美元(占售价 29%)

**海外仓成本**:

- 仓储费(月):0.3 美元
- 配送费:4.5 美元(UPS Ground)
- 包装费:0.5 美元
- 月度总成本:5.3 美元(占售价 21%)

海外仓成本低 8 个百分点,但转化率可能低 20%-30%(无 Prime 标志)。综合利润可能持平,需要根据实际数据测算。

## 海外仓选择标准

选择海外仓时考察:

- 位置:靠近主要市场或 FBA 仓库,方便调拨。
- 系统对接:支持 API 对接亚马逊、eBay 等平台,自动同步订单。
- 处理能力:日均处理订单量、峰值处理能力(旺季)。
- 退货处理:是否提供检查、翻新、拍照服务。
- 费用透明:仓储费、操作费、配送费、退货处理费是否明码标价。
- 保险:是否提供库存保险,丢货赔偿标准。
